🎮 TicTacToe (Legacy Python Project)

A collection of TicTacToe implementations built in Python using NumPy and Pygame.
This project was originally developed a long time ago and reflects an early stage in my programming journey.

⚠️ This is a legacy project and is not actively maintained.

📌 Overview

This repository contains multiple versions of TicTacToe, including:
	•	A graphical version with a bot opponent
	•	A two-player local version
	•	A console-based implementation using NumPy

Due to the lack of version control during development, the repository contains numerous experimental files, backups, and test scripts.

⚠️ Project Status
	•	Maintenance: ❌ Not maintained
	•	Code Quality: ⚠️ Unstructured / experimental
	•	Compatibility: ⚠️ May not fully work on modern Python versions

Known issues:
	•	No type annotations
	•	Minimal code organization
	•	Debug print statements may still be present
	•	Some files may be redundant or unused

🧰 Tech Stack
	•	Python (originally written for an older version)
	•	NumPy
	•	Pygame

Install dependencies:

pip install numpy pygame

📂 Key Files

Many files in this repository are not relevant. Focus on the following:

bigMain.py
	•	Primary implementation with a bot opponent
	•	Built using Pygame
	•	Most feature-complete version

main.py
	•	Two-player local mode
	•	Built using Pygame

test.py
	•	Console-based version
	•	Uses NumPy only (no GUI)
	•	Simplest way to run the game

superTicTacToe.py
	•	Experimental extended version
	•	Incomplete and abandoned

📁 Assets / Sources

The project depends on external assets located in a sources folder.

⚠️ You must download/include the sources folder for Pygame versions to run correctly.

▶️ Getting Started

Clone the repository:

git clone 
cd 

Run one of the versions:

GUI Versions (Pygame)

python main.py

or

python bigMain.py

Console Version

python test.py
